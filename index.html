Conceitos básicos de HTML

Construção de texto

Para poder construir um texto básico em HTML você precisa ter noção do que os seguintes comandos fazem:

Os seguintes códigos terão que estar entre "<....>"

  1. h1,h2,h3... = Criador de títulos e substítulos
  2. p = paragráfos para textos e explicações
  3. hr = Faz com que aja uma linha como a de baixo por exemplo

Para poder editar palavras no meio do seu texto,você pode usar os seguintes comandos:

  • strong = deixar em negrito
  • i = enclinar o texto
  • small = o texto pequeno
  • u = sublinhar o texto
  • sub = faz com que destaque o texto assim
  • sup = faz com que destaque o texto assim
  • blockquote = uma citação de algum site ou livro
    Este é um exemplo de blockquote
  • del = faz com que risque o texto
Há muitos outros comandos mas estes são alguns básico que já pode elaborar a sua escrita.
